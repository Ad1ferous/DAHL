# Даль — планировщик поездок (прототип)

Сервис помогает путешественникам превратить список желаемых мест в оптимальный маршрут с учётом личных предпочтений.

## Быстрый старт

### 1. Клонировать репозиторий
bash
git clone https://github.com/AlexTilta/DAHL.git
cd DAHL

### 2. Запустить бэкенд (Python + FastAPI)

cd backend
python -m venv venv
venv\Scripts\activate      # Windows
pip install -r requirements.txt
uvicorn main:app --reload --port 8002

Сервер будет доступен на http://localhost:8002

### 3. Запустить бэкенд (Python + FastAPI)

cd frontend
npm install
npm run dev

Приложение откроется на http://localhost:5173

### Основные функции прототипа
Регистрация и вход пользователя
Создание плана поездки (добавление мест вручную или через поиск)
Оптимизация порядка мест (жадный алгоритм с учётом расстояний)
Готовые маршруты для быстрого старта
Личный кабинет с сохранением всех планов

### Структура проекта
DAHL/
  backend/          # Серверная логика
    main.py         # Все эндпоинты
    venv/           # Виртуальное окружение
  frontend/         # Клиентская часть
    src/
      pages/        # Компоненты страниц
      styles.css    # Общие стили
  data/             # Тестовые данные 
  README.md

### Примечания
Все данные хранятся в оперативной памяти сервера. После перезапуска бэкенда требуется повторная регистрация.
Для работы карт используется временный API-ключ Яндекс.Карт. При необходимости его можно заменить в файле PlanPage.jsx.
Порт бэкенда: 8002, фронтенда: 5173 (по умолчанию).
