# Даль — планировщик поездок (прототип)

Сервис, который помогает путешественникам превратить разрозненный список желаемых мест в логичный и выполнимый маршрут с учётом личных предпочтений и времени работы.

## Быстрый старт

### 1. Клонирование репозитория
``` bash
git clone https://github.com/Ad1ferous/DAHL.git
cd DAHL ```

### 2. Запуск бэкенда (Python + FastAPI)
``` bash
cd backend
python -m venv venv
venv\Scripts\activate      # Windows
pip install -r requirements.txt
uvicorn main:app --reload --port 8002 ```

Сервер будет доступен по адресу http://localhost:8002.

### 3. Запуск фронтенда (React + Vite)
bash
cd frontend
npm install
npm run dev
Приложение откроется на http://localhost:5173.

### Основные функции прототипа
Регистрация и вход пользователя

Создание плана поездки (добавление мест вручную или через поиск по локальной базе)

Оптимизация порядка мест с помощью жадного алгоритма (учёт расстояний по прямой)

Готовые маршруты для быстрого старта

Личный кабинет с сохранением всех планов

### Используемые технологии
Бэкенд	Python, FastAPI, Uvicorn, Geopy
Фронтенд	React, Vite, React Router, Axios
Карты	Яндекс.Карты JavaScript API
Оптимизация	Жадный алгоритм с расчётом расстояний (Geopy)
Данные	Временное хранение в памяти (in-memory)

### Примечания
Все данные (пользователи, планы, места) хранятся в оперативной памяти сервера. После перезапуска бэкенда потребуется повторная регистрация.

Для работы карт используется временный API-ключ Яндекс.Карт, прописанный в файле PlanPage.jsx. При необходимости замените его на собственный ключ.

Порт бэкенда по умолчанию: 8002, порт фронтенда: 5173.
